1. 시멘틱 태그(Semantic tag)
- HTML5에서 도입된 태그로, 웹페이지의 구조와 의미를 명확하게 표현하는 태그
- Ex. <header>, <nav> ....
- 위의 태그들을 사용하면 브라우저나 검색 엔진, 스크린 리더 등이 페이지의 각 영역이 어떤 역할을 하는지 쉽게 이해할 수 있다.
- 웹 접근성과 검색 엔진 최적화(SEO)

- <header> : 머리말 영역
- <nav> : 네비게이션 메뉴 - 다른 페이지나 섹션으로 이동
- <main> : 주요 콘텐츠
- <section> : 큰 내용 중 내용 구분
- <article> : 내용을 전체적으로 감싸주는 영역 ex. 블로그 글, 뉴스기사 ~ 독립적 콘텐츠
- <aside> : 사이드바(주로 광고영역 or 메뉴)
- <footer> : 바닥글
- <figure> : 이미지 & 표 등을 감싸는
- <figcaption> : <figure>안의 콘텐츠에 대한 설명 글
- <mark> : 강조
- <time> : 시간 or 날짜
