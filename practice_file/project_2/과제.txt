메인 페이지와 시간표 페이지 만들기

index.html : 메인 화면
    간단한 자기소개 문장 작성
    timetable.html로 이동하는 하이퍼 링크 만들기

timetable.html : 시간표 페이지
- <table>을 사용하여 시간표 작성(나의 주간 시간표)
- 요일별 구조
- colspan, rowspan 중 하나 이상 필수 사용
- <thead>, <tbody> 구조 구분
- CSS 사용하여 테두리, 정렬, 배경색, 셀 간격 등 꾸미기 (외부)
- <caption> 태그가 무엇인지 알아보고 적용하기

제출 방법
슬렉 과제 채널에 index.html(도메인 주소) 링크하기
